## BreadCrumb

会读取主题配置中的以下字段

- `breadcrumbIcon`: 是否显示图标(默认显示)
- `breacurmb`: 是否启用路径导航
- `iconPrefix`: 图标前缀，默认为 `icon-`，设置 `false` 或 `''` 取消前缀，也可设置其他值。
