vuepress-theme-hope

一个轻量的 Vuepress 主题

使用说明 →

主题色切换

支持自定义主题色并允许用户在预设的主题颜色之间切换

夜间模式

可以在官方主题与一套酷炫的夜间模式间自由切换

浏览量与评论

配合 Valine 与 Vssue 来开启阅读量统计与评论支持

默认主题的增强

在默认主题基础上增强了一系列功能，包括路径导航、页脚支持、作者显示等

图标支持

贯穿整个主题的图标支持

# vuepress-theme-hope 是一个轻量主题

# 使用主题

借助 Vuepress 强大的主题系统，启用主题从未如此简单。

# 安装

npm i -D vuepress-theme-hope

# 使用

// .vuepress/config.js

module.exports = resolve({
  // your config here
